    #Deklarasi Array

    Array adalah satu set lokasi memori berurutan yang digunakan untuk menyimpan data

    • Setiap item dalam array disebut sebuah element
    • Jumlah element dalam array disebut juga dimensi array.

    Syntax :

    tipe_data nama_array[ukuran];

    Contoh :

                            	
// char adalah tipe data, name adalah "nama array", 25 adalah ukuran array
char name[25];
int mhs[50];
float panjang[50];

Student stud[100];
// Jangan khawatir, Student adalah tipe data pendefinisian user
// Kita akan belajar tentang pendefinisian user di tahan berikut nya

    #Initialisasi Array

    Ada banyak cara untuk menginitialisasikan array:

    Sebuah array dapat diinisialisasikan dalam sebuah deklarasi dengan menulis daftar koma septara nilai yang di lampirkan dalam tanda kurung diikuti tanda yang sama

                       			
int number[8] = {3, 5, 3, 7, 1, 12, 10, 121};

    Ukuran dapat ditemukan dari isi list

    Jika ukuran dihilangkan, compiler menggunakan jumlah dalam list

                       			
int number[] = {3, 5, 3, 7, 1, 12, 10, 121};

    Tanpa Nilai

    jika sebuah array tanpa di inisialisasikan , nilai tidak terdefinisi

                       			
char name[20];

    Inisialisasi Karakter

    Karakter array dapat di inisialisasi dengan menambahkan kutip 2

                       			
char name[] = "Ismail";
// Ukuran array nya adalah 6
